Declaracion

puedes declarar una variable utilizando las palabras clave let, const o var:

Var: Ámbito: Global o de función Características: Puede ser redeclarada y reasignada. No respeta el ámbito de bloque.

Ejemplo


                function ejemploVar() {
                    if (true) {
                    var nombre = "Juan";
                    console.log(nombre); // "Juan"
                    }
                    console.log(nombre); // "Juan" (a pesar de estar fuera del bloque)
                    }
                    
                    ejemploVar();

Explicación

Dentro del bloque if, declaramos var nombre = "Juan";. Aunque está dentro de un bloque {}, var no respeta el ámbito de bloque, sino el de la función. Por eso, nombre es accesible fuera del bloque if, pero sigue siendo parte de la misma función ejemploVar Problema potencial: Si hay otra variable nombre en otro bloque, puede causar conflictos, ya que var no limita el ámbito.

Let: Tiene un ámbito de bloque , lo que significa que solo es accesible dentro del bloque {}donde fue declarado. let Fue introducido en ES6 como una mejora sobre var.

Ejemplo


                function ejemploLet() {
                    if (true) {
                      let edad = 25; // Declarada con `let`, solo accesible dentro del bloque if
                      console.log(edad); // 25 (dentro del bloque if)
                    }
                    // console.log(edad); // Error: `edad` no está definida fuera del bloque if
                  }
                  
                  ejemploLet();

Explicación

Dentro del bloque if, declaramos let edad = 25; al usar let, la variable queda limitada al bloque donde se declara. Esto ayuda a evitar problemas donde el mismo nombre de variable podría usarse en diferentes bloques.

const: Ámbito: de bloque Características: se usa para declarar variables que no deben cambiar de valor . Aunque el valor de la variable no puede cambiar, si es un objeto o array , sus propiedades pueden ser modificadas.

Ejemplo


                function ejemploConst() {
                    const pi = 3.1416; // Declarada como constante
                    console.log(pi); // 3.1416
                  
                    // pi = 3.14; // Error: No se puede reasignar
                  
                    const persona = { nombre: "Ana", edad: 30 }; // Un objeto declarado como constante
                    persona.edad = 31; // Modificamos una propiedad del objeto
                    console.log(persona); // { nombre: "Ana", edad: 31 }
                  }

Explicación

const pi = 3.1416;Crea una llamada constante pi, cuyo valor no puede cambiar. Intentar reasignar pigenera un error. Sin embargo, para objetos o matrices declarados como const, puedes modificar sus propiedades internas , como cambiar edadde 30 a 31. Úsalo cuando estés seguro de que el valor no debe cambiar,

ambito de variables

Cuando declaras una variable fuera de cualquier función, se denomina variable global, porque está disponible para cualquier otro código en el documento actual. Cuando declaras una variable dentro de una función, se llama variable local, porque solo está disponible dentro de esa función.

Ejemplo

Variable global


                    if (true) { var x = 5;
                            }
                   console.log(x); // x es 5

variable local


                    if (true) { let y = 5;
                      }
                    console.log(y); // ReferenceError: y no está definida

Tipos de Dtaos

El último estándar ECMAScript define ocho tipos de datos: los siete primeros serian datos primitivos

  • Number

    • almacena valores numericos, enteros o decimales

  • BigInt

    • valores numericos enteros grandes

  • String

    • secuencia de caracteres, como texto

  • Boolean

    • true,false

  • Null

    • la variable no tiene valor o esta vacia

  • Undefined

    • variable declarada sin valor asignado

  • Symbol

    • identificador unico utilizado para propiedades de objetos

  • Object

    • Un objeto tiene propiedades asociadas a él. Una propiedad de un objeto se puede explicar como una variable adjunta al objeto. Las propiedades de un objeto básicamente son lo mismo que las variables comunes, excepto por el nexo con el objeto. Las propiedades de un objeto definen las características del objeto. Accedes a las propiedades de un objeto con una simple notación de puntos:. objectName.propertyName; o tambien podrias hacerlo a traves de corchetes, var myCar = { make: "Ford", model: "Mustang", year: 1969, };

  • array

    • Los arrays son objetos similares a una lista cuyo prototipo proporciona métodos para efectuar operaciones de recorrido y de mutación. Tanto la longitud como el tipo de los elementos de un array son variables. para acceder a estos elementos lo haremos a traves de su indice, teniendo siempre en cuenta que el primer valor por defecto es cero let primero = frutas[0]; // Manzana let ultimo = frutas[frutas.length - 1]; // Banana

asignacion de variables

La asignación de valores a una variable se realiza con el operador =:

Ejemplo


             let edad = 30; // asigna 30 a la variable edad//
            let nombre= "juan" // asigna "juan" a la variable nombre //

reasignacion de variables, con let y var Puedes cambiar el valor de una variable en cualquier momento.

Ejemplo


            let color = "rojo";
            console.log(color); // "rojo"

            color = "azul"; // Reasignamos un nuevo valor
            console.log(color); // "azul"

const: No puedes reasignar un valor después de declarar una variable con const. Aunque const no permite la reasignación de un objeto o array, sí puedes modificar sus propiedades internas .

Ejemplo


            const persona = { nombre: "Ana", edad: 25 };
            persona.edad = 26; // Modificamos la propiedad `edad`
            console.log(persona); // { nombre: "Ana", edad: 26 }

Asignación por Valor y Asignación por Referencia

Por valor: Los tipos primitivos (número, cadena, booleano, undefinido, nulo) se copian directamente. Por referencia: Los objetos (incluidos arrays y funciones) comparten una referencia en memoria.

Ejemplo de asignacion por valor


        let a = 10;
        let b = a; // Se copia el valor de `a`

        b = 20; // Cambiamos el valor de `b`, pero `a` no se ve afectado
        console.log(a); // 10
        console.log(b); // 20

Ejemplo de asignacion por referencia


        let objeto1 = { nombre: "Ana" };
        let objeto2 = objeto1; // Ambos apuntan al mismo objeto en memoria

        objeto2.nombre = "Carlos"; // Cambiamos el nombre a través de objeto2
        console.log(objeto1.nombre); // "Carlos" (objeto1 también cambió)

Desestructuración en la Asignación

La desestructuración es una forma de extraer valores de objetos o arrays directamente en variables.

ejemplo con arrays


        let numeros = [10, 20, 30];
        let [a, b, c] = numeros;

        console.log(a); // 10
        console.log(b); // 20
        console.log(c); // 30

ejemplo con object


        let persona = { nombre: "Ana", edad: 25 };
        let { nombre, edad } = persona;

        console.log(nombre); // "Ana"
        console.log(edad); // 25

instancia de una variable

El concepto de instancia de una variable se refiere a la creación de un valor único o de una estructura de datos que se puede usar. Aunque el término "instancia" suele asociarse más con objetos y clases (en programación orientada a objetos), en JavaScript una variable es una "instancia" de un valor en particular en el momento en que se le asigna un valor.

Ejemplo


        let persona ={nombre: "juan", edad: 30};
