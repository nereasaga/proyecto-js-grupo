Variables

¿Que son las variables?

Una variable en JavaScript es un contenedor de datos que se utiliza para almacenar valores, como números, cadenas de texto, objetos, entre otros. En otras palabras, puedes pensar en ella como una caja con un nombre, donde dentro puedes guardar un valor que puedes usar y modificar durante la ejecución de un programa.

Tipos de Variables

  • undefined

    • Cuando una variable se ha declarado, pero no se le ha asignado un valor.

  • String

    • Contiene secuencias de caracteres, como texto.

  • Boolean

    • Representa valores verdadero o falso

  • number

    • Almacena valores numéricos, ya sean enteros o decimales.

  • bigint

    • Para manejar numeros enteros grandes

  • symbol

    • Identificador unico utilizado para propiedades de objetos

  • null

    • Indica variable sin valor o vacia

  • funcion

    • Es un bloque de codigo que puede ser llamado

  • object

    • Estructura de datos que puede contener propiedades y métodos.

  • arrays

    • son objetos similares a una lista cuyo prototipo proporciona métodos para efectuar operaciones de recorrido y de mutación. Tanto la longitud como el tipo de los elementos de un array son variables.

de estos los siete primeros son considerados primarios

Tipado dinamico

JavaScript es un lenguaje de tipado dinámico. Las variables en JavaScript no están asociadas directamente con ningún tipo de valor en particular, y a cualquier variable se le puede asignar (y reasignar) valores de todos los tipos

number

                    let variable = 42; // Inicialmente es un número
                    console.log(variable);
string

                    variable = "Hola"; // Ahora es un string
                    console.log(variable);
boolean

                    variable = "true"; // Ahora es un booleano
                    console.log(variable);

Declaracion de variables

puedes declarar una variable utilizando las palabras clave let, const o var:

let:

Se utiliza para declarar variables cuyo valor puede cambiar.


               let nombre ="pedro";
                nombre="Carlos" // el valor puede ser reasignado
const:

Se utiliza para declarar variables cuyo valor no puede cambiar (constantes).


              const pi = 3.1416;
              consol.log(pi) // pi= 3.14; // esto daria error, porque el valor no puede cambiar//
var (antiguo):

Era el método tradicional, pero hoy se usa menos debido a su comportamiento menos predecible en cuanto a alcance.


              var edad= 20;
               edad=21 // se puede reasignar//

Ambito de variables

Cuando declaras una variable fuera de cualquier función, se denomina variable global, porque está disponible para cualquier otro código en el documento actual. Cuando declaras una variable dentro de una función, se llama variable local, porque solo está disponible dentro de esa función.

Ejemplo

global

                let mensajeGlobal = "Soy una variable global";
                console.log(mensajeGlobal);
local

               let mensajeLocal= "soy una variable local";
                console.log(mensajeLocal);

Instancia de una variable

Instanciar objetos en JavaScript se refiere al proceso de crear un nuevo objeto basado en una plantilla o molde predefinido, que generalmente se conoce como una “clase”, Para hacer esto, JavaScript proporciona una palabra clave especial: new.

Ejemplo


            class Libro {
              constructor(titulo, autor, anioPublicacion) {
                  this.titulo = titulo;
                  this.autor = autor;
                  this.anioPublicacion = anioPublicacion;
              }
              obtenerDescripcion() {
                  return `El libro "${this.titulo}" fue escrito por ${this.autor} en el año ${this.anioPublicacion}.`;
              }
          }
          let miLibro = new Libro('Cien años de soledad', 'Gabriel García Márquez', 1967);
          console.log(miLibro.obtenerDescripcion());
