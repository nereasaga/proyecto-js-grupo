Objetos JSON en JavaScript

¿Qué es un objeto JSON?

JSON (JavaScript Object Notation) es un formato ligero y de texto para almacenar e intercambiar datos. Es utilizado principalmente para representar datos estructurados de manera sencilla, en formato de clave-valor, es fácil de leer y escribir tanto para los humanos como para las máquinas. 

               
                    {"nombre": "Juan",
                    "edad": 30,
                    "esEstudiante": false,
                    -----------anidado------------
                    "dirección": {
                    "calle": "Calle Falsa 123",
                    "ciudad": "Bilbao",},
                    ------------arreglo------------
                    "teléfonos": ["123456789", "987654321"]
                }

Explicación: En este objeto, tenemos tres propiedades:

  • "nombre": Es una cadena de texto con el valor "Juan".
  • "edad": Es un número con el valor 30.
  • "esEstudiante": Es un valor booleano (true o false), en este caso es true

Ejemplo 3: Objeto JSON con un arreglo

Este objeto JSON incluye un arreglo (lista) dentro de la propiedad "telefonos":


            {"nombre": "Juan",
            "edad": 30,
            "esEstudiante": false,
            -----------anidado------------
            "dirección": {
              "calle": "Calle Falsa 123",
              "ciudad": "Bilbao",},
            ------------arreglo------------
            "teléfonos": ["123456789", "987654321"]
          }

Explicación: En este caso, la propiedad "telefonos" es un arreglo que contiene dos números de teléfono como cadenas de texto.
