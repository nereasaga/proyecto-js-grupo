Aritméticos

Estos operadores realizan operaciones matemáticas como suma, resta, multiplicación, etc.


let a = 10;
let b = 3;
console.log(a + b); // Suma: 13
console.log(a - b); // Resta: 7
console.log(a * b); // Multiplicación: 30
console.log(a / b); // División: 3.333...
console.log(a % b); // Módulo: 1

Asignación

Se usan para asignar valores a las variables. Algunos permiten modificar el valor de una variable con una operación.


let x = 10; // Asigna 10 a x
x += 5; // Suma 5 a x (x = x + 5)
console.log(x); // 15

Comparación

Comparan dos valores y devuelven true o false según el resultado de la comparación.

Igual que

let a = 5;
let b = 10;
console.log(a === b);

Diferente de

let a = 5;
let b = 10;
console.log(a !== b);

Menor que

let a = 5;
let b = 10;
console.log(a < b);

Mayor o igual que

let a = 5;
let b = 10;
console.log(a >= b);

Lógicos

Evalúan condiciones múltiples.


let a = 10;
let b = 3;

console.log(a + b); // Suma: 13
console.log(a - b); // Resta: 7
console.log(a * b); // Multiplicación: 30
console.log(a / b); // División: 3.333...
console.log(a % b); // Módulo: 1

Cadena

Trabajan con texto (strings).


let firstName = "John";
let lastName = "Doe";

console.log(firstName + " " + lastName); // Concatenación: John Doe
console.log(firstName.length); // Longitud de la cadena: 4

Incrementales y Decrementales

Aumentan o disminuyen valores.


let count = 0;

count++; // Incrementa en 1
console.log(count); // 1
count--; // Decrementa en 1 console.log(count); // 0

Operadores Bit a Bit

Realizan operaciones a nivel binario (avanzado).


console.log(5 & 1); // AND binario: 1
console.log(5 | 1); // OR binario: 5

Operadores especiales

Tienen usos más específicos


let person = { name: "Alice" };
console.log(typeof person); // Tipo: object
console.log(person instanceof Object); // ¿Es instancia de Object?: true
                
let age = 18;
console.log(age >= 18 ? "Adulto" : "Menor"); // Operador ternario: Adulto
