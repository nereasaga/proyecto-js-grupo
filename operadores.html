Aritméticos

Estos operadores realizan operaciones matemáticas como suma, resta, multiplicación, etc.

Suma

let a = 10;
let b = 3;
console.log(a + b)
Resta

let a = 10;
let b = 3;
console.log(a - b)
Multiplicacion

let a = 10;
let b = 3;
console.log(a * b)
Division

let a = 10;
let b = 3;
console.log(a / b)
Modulo

let a = 10;
let b = 3;
console.log(a % b)

Asignación

Se usan para asignar valores a las variables. Algunos permiten modificar el valor de una variable con una operación.

var

var six = 5;
console.log(var)
let

let six = 5;
console.log(six)
const

const six = 5;
console.log(six)

Comparación

Comparan dos valores y devuelven true o false según el resultado de la comparación.

Igual que

let a = 5; 
let b = 10;

console.log(a === b);
Diferente de

let a = 5; 
let b = 10;

console.log(a !== b);
Menor que

let a = 5;
let b = 10;

console.log(a < b);
Mayor o igual que

let a = 5;
let b = 10;

console.log(a >= b);

Lógicos

Evalúan condiciones múltiples.

if both were true

let a = true;
let b = false;

console.log(a && b)
if one of them is true

let a = true;
let b = false;

console.log(a || b)
print the oppsite of the value
let a = true;
let b = false;

console.log(!a)
print the opposite of the value

let a = true;
let b = false;

console.log(!b)
String

Trabajan con texto (strings).


let firstName = "John";
let lastName = "Doe";

console.log(firstName + lastName);


console.log(firstName + " " + lastName);

Incrementales y Decrementales

Aumentan o disminuyen valores.

Incrementales

Incrementa en 1


let count = 0;
console.log(++count);
Decrementales

Decrementa en 1


            let count = 0;
            console.log(--count);

Operadores Bit a Bit

Realizan operaciones a nivel binario (avanzado).


console.log(5 & 1); // AND binario: 1
                     

console.log(5 | 1); // OR binario: 5

Operadores especiales

Tienen usos más específicos


            let person = { name: "Alice" };
            console.log(typeof person); // Tipo: object
            console.log(person instanceof Object); // ¿Es instancia de Object?: true
            

            let age = 18;
            console.log(age >= 18 ? "Adulto" : "Menor"); // Operador ternario: Adulto
